# munin-aws
## Munin AWS ElasticSearch plugin
In your /etc/munin/plugin-conf/munin-node
```
[elasticsearch-*]
env.host [host]
env.index [index]
```
Suggest
```
/usr/share/munin/plugins/elasticsearch- suggest
```

Example Docs:
```
ln -s /usr/share/munin/plugins/elasticsearch- elasticsearch-docs
```
